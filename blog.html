Important FAQ

What is the Document Object Model(DOM)?

Dom is crating a tree type model of all the html data.By using the DOM we can manupulate,add and update the html elements.

How do you select an element from the DOM?

There are many ways to select and element from Dom:
1. getElementById()
2.getElementsByClassName()
3.getElementsByTagName()
4.querySelector()
5.querySelectorAll()

What is event delegation in the context of the DOM,and why is it useful?

1.Event Delegation is a technique where a parent element handles events triggered by its child elements.
2. It is useful for improving performance, especially with a large number of child elements or dynamically added elements.
3. It simplifies code by reducing the number of event listeners and helps manage events centrally.

How do you manipulate an element's attributes and styles using the DOM?

we can manipulate the element in serveral ways, 1.Attributes: getAttribute(), setAttribute(), and removeAttribute()
2. Styles: style and classList. etc
